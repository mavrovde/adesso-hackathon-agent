The Brake

Two layers of safety — deterministic first

🛑 Hard Blocks — PreToolUse

Fires before the model. Cannot be bypassed by prompt injection.

  • Reset password on FROZEN or UNDER_INVESTIGATION account
  • Auto-resolve a P1 ticket
  • Any tool call containing known prompt-injection patterns
  • PII exfiltration attempt (send to external endpoint)
  • Write action on a VIP account without human approval

⬆ Soft Escalation Rules

Evaluated by the coordinator after classification.

  • Classification confidence < 0.75
  • User is VIP (C-Level / Board / whitelist)
  • Request mentions legal, GDPR, Datenschutz, audit
  • Estimated business impact > €10,000
  • Request language is not English or German
def pre_tool_use_hook(tool_name, tool_input, context) → dict | None: # Hard block: frozen account if tool_name == "reset_user_password": if context["account_status"] in {"FROZEN", "UNDER_INVESTIGATION"}: return {"block": True, "reason": "Account frozen — write actions hard-blocked"} # Hard block: P1 auto-resolve if tool_name == "resolve_ticket" and context["priority"] == "P1": return {"block": True, "reason": "P1 requires human approval"} # Prompt injection scan (any tool) for pattern in PROMPT_INJECTION_PATTERNS: if pattern in str(tool_input).lower(): return {"block": True, "reason": f"Injection detected: '{pattern}'"}

The Mandate — Legal & Compliance

Every boundary documented before a single line of code

✓ Agent Acts Alone

  • Classify all inbound requests
  • Route to team queue (confidence ≥ 0.75)
  • Auto-reset password (active, unflagged, SSO)
  • Block adversarial / injection input
  • Create ITSM ticket with reasoning log

⬆ Agent Escalates

  • Confidence < 0.75 (clarification first)
  • VIP: C-Level, Board, whitelist
  • Legal / GDPR / compliance mention
  • Impact estimate > €10,000
  • Unknown language (not EN/DE)
  • 3 failed validation retries

✕ Agent Never Touches

  • Lock, suspend, or delete accounts
  • Transmit PII outside ITSM context
  • Act on FROZEN / UNDER_INVESTIGATION
  • Override a prior human decision
  • HR, payroll, or ERP systems
  • Bulk operations (> 1 account)

Auditability: Every routing decision produces a structured log with category · confidence · routing_target · reasoning · escalated · retry_count · timestamp · request_id. Immutable. 12-month retention. No PII in reasoning log. Fully replayable.
